# EC site development

長野県にある小さな洋菓子店「ながのCAKE」の商品を通販するためのECサイト開発。

# サイト概要
* 通販では注文に応じて製作する受注生産型としている
* 現在通販での注文量は十分に対応可能な量のため、1日の受注量に制限は設けない
* 送料は1配達につき全国一律800円
* 友人や家族のプレゼントなど、注文者の住所以外にも商品を発送できる
* 支払方法はクレジットカード、銀行振込から選択できる
